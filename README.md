
# ❤️ HeartSync

HeartSync is a relationship compatibility calculator built using HTML, CSS, and JavaScript.

Instead of generating a random score, the application compares two people based on their preferences, personality traits, music taste, and birth date information to calculate an overall compatibility score.

----------------

## FEATURES

* Side-by-side partner comparison
* Food preference matching
* Music preference comparison
* Personality compatibility analysis
* Birth date compatibility scoring
* Zodiac sign matching bonus
* Compatibility breakdown by category
* Shared interests display
* Personalized compatibility messages
* Responsive design for desktop and mobile

-----------------

### UI DESIGN

The interface is designed around a side-by-side comparison layout, allowing both partners' information to remain visible throughout the process.

Design Highlights

* Glassmorphism-inspired cards
* Gradient background
* Responsive layout
* Rounded corners and soft shadows
* Smooth transitions and animations
* Mobile-friendly experience

The goal was to create a clean and modern user experience while keeping the application lightweight and easy to use.

-----------------

### TECHNOLOGIES USED

* HTML
* CSS
* JavaScript
* CSS Grid


No frameworks or backend services were used.
